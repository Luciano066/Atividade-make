# Projeto Shell Script

Script em Bash para organização automática de arquivos em um projeto de hardware digital.

## 📂 Estrutura

- src/ → arquivos principais (.v)
- tb/ → testbench
- include/ → headers (.vh)
- scripts/ → scripts (.tcl, .do, .sh)
- docs/ → documentação

## ⚙️ Como usar

```bash
./organiza_projeto.sh
